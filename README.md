# 三角洲代练工作室管理系统

## 🎯 项目简介

这是一个专为游戏代练工作室打造的全流程订单管理与运营平台。系统覆盖从客户下单、客服发单、打手接单、订单执行、完结结算到提现审核的完整业务闭环。

## ✨ 核心功能

### 📦 订单全生命周期管理
- 客户下单
- 客服审核与发单
- 打手抢单与接单
- 订单执行与进度跟踪
- 订单验收与结算
- 提现申请与审核

### 💼 企业微信集成
- 自动推送代练单到派单群
- 打手接单消息自动识别
- 订单状态实时通知
- 消息存档与日志

### 👥 人员管理
- 多角色权限管理（管理员、客服、打手、财务）
- 打手绩效统计与排行榜
- 信誉分体系
- 客户关系管理

### 💰 财务对账
- 资金流水管理
- 日/周/月财务报表
- 提现审批流程
- 账户余额管理

### ⚠️ 风险管控
- 订单逾期预警
- 打手信誉预警
- 财务异常检测
- 操作日志审计

## 🚀 快速开始

### 环境要求
- Node.js >= 18
- npm 或 pnpm

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

这将同时启动：
- 前端开发服务器: http://localhost:5173
- 后端 API 服务器: http://localhost:3001

### 登录信息

默认账号：
- 用户名：`admin`

## 🔧 企业微信配置

### 配置步骤

1. **创建企业微信应用**
   - 登录企业微信管理后台
   - 进入"应用管理"创建自建应用
   - 设置应用权限和可见范围

2. **获取应用凭证**
   - AgentId：在应用详情页获取
   - AgentSecret：在应用详情页获取
   - CorpId：在"我的企业"页面获取

3. **创建派单群**
   - 在企业微信中创建派单群聊
   - 将打手和客服加入群聊
   - 获取群聊 ID

4. **配置系统**
   - 在系统设置页面填入上述信息
   - 测试连接确保配置正确

### 消息模板

系统支持自定义消息模板，变量包括：
- `{game}` - 游戏名称
- `{content}` - 代练内容
- `{price}` - 代练价格
- `{deadline}` - 截止时间
- `{requirements}` - 特殊要求

## 📝 开发说明

### API 接口

所有 API 响应格式统一：

```json
{
  "success": true,
  "data": {},
  "message": "操作成功"
}
```

分页响应：

```json
{
  "success": true,
  "data": {
    "items": [],
    "total": 100,
    "page": 1,
    "pageSize": 10,
    "totalPages": 10
  }
}
```

### 订单状态流转

```
pending → verified → published → assigned → in_progress → completed
                                    ↓
                               cancelled
                                    ↓
                               disputed
```

## ⚠️ 注意事项

1. **数据安全**：生产环境请务必修改 JWT Secret 和加密密钥
2. **企业微信**：首次使用需要配置企业微信应用凭证
3. **敏感信息**：客户账号密码等敏感数据已加密存储
4. **日志审计**：关键操作会自动记录日志
